1. ZeroToMillion — игра, которая сможет окунуть в брокерскую жизнь. Мы стартуем с 0$, затем берем кредит на определенную сумму, затем совершаем операции на бирже, цель достичь 10**6$ с учетом погашения кредита.
2. Сделать банковскую систему. Для начала просто будем выдавать кредиты с определенным процентом, затем сделать этот процент реальным.
3. Создать эмулятор биржи.
4. Добавить Json, в которую записывается сохранение прогресса, а также цены товаров
5. Добавить казино как отдельный вид заработка
